CCTV Monitoring System

Sistem monitoring CCTV berbasis web untuk pengawasan lalu lintas kendaraan secara realtime menggunakan teknologi HLS (HTTP Live Streaming).

Fitur Utama:
  • Live streaming video HLS
  • Deteksi kendaraan otomatis
  • Monitoring realtime 24/7
  • Dashboard interaktif
  • Responsive design
Teknologi:
  • Backend: Flask (Python)
  • Frontend: HTML5, Bootstrap 5, JavaScript
  • Video Streaming: HLS (HTTP Live Streaming)
  • Video Player: Video.js / HLS.js
Info: Sistem ini menggunakan HLS streaming untuk memastikan kompatibilitas yang baik di berbagai browser dan perangkat.
